# THEME PACK TEMPLATE

> 用途：后续新主题包（仙班 / 后宫 / 西游等）都可以基于这份模板快速起盘。

---

## 1. 核心不变
每一套 pack 都应继承：
- roster
- task assignment
- review gate
- execution flow
- expandable roles
- Lite / Standard / Full presets

## 2. 可替换层
每一套 pack 可以替换：
- 角色命名
- 组织语言
- 头像
- 文案
- 世界观
- UI 命名与展示氛围

## 3. 必备公开资产
- `README.md`
- `pack.json`
- `PRESET_GUIDE.md`
- `ROLE_ROSTER.md`
- `ROLE_CARD_COPY.md`
- `30_SECONDS_TO_INSTALL.md`
- `CHANGELOG.md`
- `scripts/`
- `assets/`

## 4. 推荐对外母句
### 中文
安装一个组织，而不是一个 bot。

### English
Install an organization, not just a bot.

## 5. 推荐 GitHub Description 结构
A runnable AI organization pack with roster, review gates, execution roles, and modular presets — themed as <theme>.
