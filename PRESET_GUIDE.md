# PRESET GUIDE

三省六部包采用 **Full 母本 + 预设安装** 模式。

## Lite
适合：
- 第一次体验多 agent 的用户
- 想先感受朝廷闭环的人

安装角色：
- 太子
- 中书省
- 门下省
- 尚书省
- 工部

## Standard
适合：
- 想要更完整协作，但不需要全朝廷的人
- 同时在意内容、资源、推进、风险的人

安装角色：
- 太子
- 中书省
- 门下省
- 尚书省
- 工部
- 礼部
- 户部
- 兵部
- 刑部

## Full
适合：
- 想完整拥有一座 AI 朝廷的人
- 想自己决定保留谁、裁掉谁的人

安装角色：
- 太子
- 中书省
- 门下省
- 尚书省
- 吏部
- 户部
- 礼部
- 兵部
- 刑部
- 工部
- 翰林院

## 安装命令

### Lite
```bash
bash scripts/install-pack.sh --preset lite
```

### Standard
```bash
bash scripts/install-pack.sh --preset standard
```

### Full
```bash
bash scripts/install-pack.sh --preset full
```

## 为什么这样设计

因为我们相信：

> 应该先把整座朝廷做出来，再允许用户自己做减法。

这比一开始就给用户一个删减过、想象力不足的版本更强。
