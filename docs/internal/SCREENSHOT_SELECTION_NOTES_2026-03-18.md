## 当前口径差异

截图整体更偏：
- 员工
- 招聘
- 市场
- roster
- dashboard

而当前文案整体更偏：
- 三省六部
- 朝廷
- 官员
- 上朝
- 班底

### 这不一定是坏事
它说明我们其实有两条可走路径：

#### 路径 A：继续强化世界观
让截图作为“现实系统壳”，文案继续走三省六部叙事。

#### 路径 B：往现代组织产品靠
把“朝廷”更多退到灵感层，公开展示更偏“员工组织系统”。

### 当前建议
首发阶段更稳的是：

> **保留三省六部的世界观钩子，但对外展示多使用“班底 / roster / 组织 / 员工系统”这类更容易被理解的词。**

这样既不丢特色，也不至于让真实 UI 和营销语言完全打架。
