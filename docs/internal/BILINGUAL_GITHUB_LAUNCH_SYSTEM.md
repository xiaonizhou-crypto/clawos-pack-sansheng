## 6. 面向入门者的解释 / Beginner-friendly Explanation

### 中文
如果你是第一次接触多 agent，可以把它理解成：

- **不是一个 AI 帮你做所有事**
- 而是把不同职责拆给不同角色
- 有人先分案
- 有人先写草案
- 有人负责审核
- 有人负责真的把东西做出来

这样做的好处是，AI 不再像群聊里抢话筒，而更像一个真正的团队。

### English
If you’re new to multi-agent systems, think of it like this:

- not one AI doing everything,
- but different roles handling different parts of the job,
- one role triages,
- one drafts,
- one reviews,
- one actually ships.

The result is less “every bot talking at once” and more “a team with structure.”
