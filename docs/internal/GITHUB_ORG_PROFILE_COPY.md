## 2. 组织主页长版介绍 / Long Profile Intro

### 中文
三省六部是第一套已经落地的主题组织包。它证明了一件更大的事：AI 不只是一个个并排说话的 bot，也可以被组织成有班底、有审核门、有执行链、还能继续扩编的系统。

而这只是开始。后面我们还会继续把同一个 organization kernel 扩展到更多世界观：西游、后宫、仙班，以及更多“能运行、能传播、能记住”的 AI 世界。

### English
三省六部 is the first landed theme pack, but it points to something bigger: AI should not have to feel like a bunch of bots talking in parallel. It can be organized into a roster, a review gate, an execution flow, and an expandable system that keeps growing.

And this is only the start. The same organization kernel can expand into more worlds next — Journey to the West, Palace systems, Celestial courts, and other runnable, memorable, and extensible AI worlds.
