## 6. GitHub 首页长介绍（120–200 字）

### 中文主版
三省六部不是一个古风皮肤包，也不是几个人设换皮。它想解决的是：为什么大多数多 agent 系统最后更像一群模型同时说话，而不像一个真正能运转的组织。这个项目把 AI 做成了一套有 roster、有审核门、有执行链、还能继续扩编补位的组织系统。你可以先装 Lite，也可以一步装满 Full，再按自己的需要留人、裁人、调编制。

### 英文版
三省六部 is not just a themed skin for multiple agents. It is an AI organization system designed to make agents feel less like parallel chatbots and more like a structured team: roster, review gates, execution roles, and expandable hiring-style growth. Start with Lite, grow into Standard, or install the full court.
