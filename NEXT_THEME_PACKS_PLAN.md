# NEXT THEME PACKS PLAN

> 目标：基于三省六部已经跑通的组织结构，为后续主题包建立统一入口与命名框架。  
> 原则：内核稳定，角色皮肤可换；先有结构，再有世界观。

---

## 1. 已经验证的稳定内核

当前三省六部已验证成立的，不只是题材，而是这些组织能力：
- roster
- task assignment
- review gate
- execution flow
- expandable roles
- Lite / Standard / Full presets

也就是说，未来真正复用的不是“三省六部”这层名字，
而是：

> **一个可运行的 AI organization pack kernel**

---

## 2. 后续主题包推荐方向

### A. 仙班 / Celestial Court
#### 气质
- 仙职
- 天庭秩序
- 神职分工
- 更高位、更庄严、更天命感

#### 适合表达
- 上承天命
- 神职编制
- 仙官体系
- 更强秩序与等级感

#### 推荐仓名
- `clawos-pack-xianban`

---

### B. 后宫 / Palace System
#### 气质
- 内廷秩序
- 亲疏关系
- 情绪张力
- 权力与协商并存

#### 适合表达
- 更强角色关系戏剧性
- 更强人物感
- 更适合做“关系驱动型协作”表达

#### 推荐仓名
- `clawos-pack-hougong`

---

### C. 西游 / Journey Pack
#### 气质
- 取经小队
- 师徒分工
- 冒险型协作
- 动态任务推进

#### 适合表达
- 小队制
- 旅途推进
- 不同角色能力互补
- 更强任务线感

#### 推荐仓名
- `clawos-pack-xiyou`

---

## 3. 命名规则建议

统一采用：
- `clawos-pack-<theme>`

例如：
- `clawos-pack-sansheng`
- `clawos-pack-xianban`
- `clawos-pack-hougong`
- `clawos-pack-xiyou`

这样好处是：
- GitHub 上一眼能看出同系列
- 未来好做列表、合集、生态页
- 更容易形成 theme-pack platform 的产品感

---

## 4. 推荐统一对外表达

### 中文
三省六部不是终点，而是第一套已经落地的主题组织包。后面我们会继续把同一个组织内核扩展到更多世界：仙班、后宫、西游，以及更多可运行的 AI 组织宇宙。

### English
三省六部 is not the end state. It is the first landed theme pack. The same kernel can expand into more worlds next — celestial courts, palace systems, Journey to the West, and more runnable AI organization worlds.

---

## 5. 推荐下一步顺序

如果只选一个最适合接在三省六部后面的方向，我建议：

### 第一优先：仙班
因为它和三省六部共享：
- 秩序感
- 官阶感
- 编制感
- 组织语言的自然延展

它最适合作为：
> **三省六部之后的第一套平滑升级主题包**

### 第二优先：西游
因为它更容易做传播，角色辨识度高。

### 第三优先：后宫
因为它戏剧性很强，但更容易走偏，需要更谨慎拿捏产品边界。
