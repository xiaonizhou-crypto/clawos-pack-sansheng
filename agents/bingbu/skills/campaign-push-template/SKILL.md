Use this skill when a task must keep moving and cannot be allowed to stall.

Default output format:

```md
# 兵部督办单
- Mission:
- Current Stage:
- Critical Node:
- Delay Risk:
- Immediate Push:
- Next Checkpoint:
```

Rules:
- Focus on movement, not commentary.
- Identify where work is stalled.
- Always define the next checkpoint.
