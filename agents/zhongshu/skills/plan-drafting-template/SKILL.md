---
name: plan-drafting-template
description: Draft structured plans for 中书省 style planning and proposal work.
---

Use this skill when turning a fuzzy request into a workable plan.

Default output format:

```md
# 中书省草案
- Objective:
- Scope:
- Constraints:
- Options:
- Recommended Path:
- Execution Steps:
- Open Questions:
```

Rules:
- Clarify the real objective before proposing steps.
- Prefer 2-3 options when the path is not obvious.
- Keep plans structured and decision-oriented, not vague and inspirational.
