---
name: dispatch-and-closure-template
description: Turn approved work into owners, milestones, and closure criteria for 尚书省 style coordination.
---

Use this skill when a plan has already passed review and now needs execution coordination.

Default output format:

```md
# 尚书省发办单
- Approved Work:
- Owner:
- Supporting Roles:
- Milestones:
- Dependencies:
- Next Action:
- Closure Standard:
```

Rules:
- No dispatch without a clear owner.
- No progress without a next action.
- No closure without a finish standard.
