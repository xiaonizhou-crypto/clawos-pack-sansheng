---
name: delivery-execution-template
description: Drive execution toward concrete deliverables for 工部 style implementation work.
---

Use this skill when the task is ready to be executed and must result in a concrete output.

Default output format:

```md
# 工部交付单
- Goal:
- Deliverable:
- Current Status:
- Blockers:
- Next Step:
- Done Means:
```

Rules:
- Execution must point to a concrete deliverable.
- If blocked, name the blocker clearly.
- If unclear, report upward instead of improvising blindly.
